
🛠 **Tech Stack**
Frontend:

React.js (with Vite)
Bootstrap 5
React Router DOM

Backend:

Node.js
Express
MongoDB 
=======

## 🚀 Features

- Three-panel role-based architecture (Admin, Company, User)
- Login authentication for users and companies
- Dynamic product listing and cart functionality
- Secure route protection based on user roles
- Clean UI with Bootstrap styling
- Company-specific product management

## ⚙️ Tech Stack

- **Frontend:**
  - React.js (with Vite)
  - Bootstrap 5

- **Backend:**
  - Node.js

Install Dependencies
cd Frontend
npm install
npm run dev

For backend (Node.js)
cd Backend
npm install
npm start
